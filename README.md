# Ecosphere-ESG-Management-platform
🌿 EcoSphere - ESG Management System

Overview of the Project

EcoSphere is the ESG Management System

The primary focus of the development was to provide an easy-to-use and convenient system of managing the organization's ESG activities. In order to avoid having to maintain it all manually, EcoSphere offers a user-friendly dashboard with the ability to control the environmental performance, employee activity, and ESG progress.

---

Key Features

- 📊 Dashboard with ESG overview
- 🌱 Environmental management
- 👥 Social and CSR activity management
- 🛡 Governance and compliance management
- 🏆 Employee challenges and rewards
- 📄 ESG reports
- ⚙️ Settings and configuration

How to Run

1. Clone the repo.
2. Open the code in VS Code.
3. Launch the frontend with Live Server.
4. Run the Spring Boot backend.
5. Link the MySQL database.
6. Browse the EcoSphere dashboard.
